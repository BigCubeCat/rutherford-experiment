# Симуляция опыта резерфорда по рассеянию альфа-частиц

Симулирует поток частиц, напраленный на атом золота.

Показывает статистику угла рассеяния альфа-частиц


![](./assets/img.png)

## Сборка
Зависимости:
- Qt6
```bash
cmake CMakeLists.txt -B ./build
make -C ./build
```
## Запуск
```bash
./build/au
```
